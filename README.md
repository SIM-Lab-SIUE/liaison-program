# Open Methods Hub

[![License: CC BY 4.0](https://img.shields.io/badge/License-CC_BY_4.0-lightgrey.svg)](https://creativecommons.org/licenses/by/4.0/)
[![Quarto Website](https://img.shields.io/badge/Built_with-Quarto-blue)](https://quarto.org)

Central course website for **MC 451 Research Methods in Mass Media** and **MC 501** at Southern Illinois University Edwardsville. The Hub integrates the textbook, R workbook, Methods Vault, and all course resources into a single site.

**Live site:** <https://sim-lab-siue.github.io/liaison-program/>

**Instructor:** Dr. Alex P. Leith (aleith@siue.edu)

---

## Site Structure

| Section | Description |
|---------|-------------|
| **Home** | Course overview, quick-start cards, ecosystem map |
| **Syllabus** | MC 451 (Spring 2026) and MC 501 (Fall 2026) |
| **Phases** | Five progressive modules: Journalist, Architect, Builder, Analyst, Publisher |
| **Workbook** | 7-chapter R workbook with narrated video walkthroughs (embedded) |
| **Textbook Preview** | Fall 2026 edition of *From Vibes to Variables* — 22-chapter outline |
| **Methods Vault** | Obsidian workspace template — download, setup, and usage guide |
| **Resources** | Software installation guides, downloadable assets, FAQ |

### The Five Phases

| Phase | Role | Focus | Weeks | Points |
|-------|------|-------|-------|--------|
| I | The Journalist | Reading journals, critical thinking, GitHub setup | 1–15 | 150 |
| II | The Architect | Software setup, archive research, project planning | 2–4 | 125 |
| III | The Builder | Ethics, theory, codebook, sampling, pilot testing | 5–12 | 250 |
| IV | The Analyst | R programming: wrangling, visualization, inference | 13–15 | 225 |
| V | The Publisher | Final web portfolio: PDF + GitHub Pages | 16–17 | 250 |

### R Workbook (embedded)

The workbook lives inside the site at `workbook/` with a docked sidebar for chapter navigation:

| Chapter | Title |
|---------|-------|
| 1 | From Vibes to Variables: Building Your Quantitative Codebook |
| 2 | Data Collection: Consistency at Scale |
| 3 | Meet R and RStudio |
| 4 | Data Wrangling: Import, Diagnose, Clean, Export |
| 5 | Describing Data: Tables and Visualizations |
| 6 | Inferencing Data: Testing Relationships |
| 7 | Publishing Your Research: Quarto Books & GitHub Pages |

Each chapter opens with a narrated video walkthrough generated by [Orator](https://github.com/SIM-Lab-SIUE/orator) using a fine-tuned F5-TTS voice clone.

## Building Locally

### Prerequisites

- [Quarto](https://quarto.org/docs/get-started/) (v1.4 or later)

### Render the site

```bash
quarto render
```

Output goes to `docs/`. Open `docs/index.html` in your browser.

### Preview with live reload

```bash
quarto preview
```
