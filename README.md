# MC Research Methods: The Liaison Program

[![License: CC BY 4.0](https://img.shields.io/badge/License-CC_BY_4.0-lightgrey.svg)](https://creativecommons.org/licenses/by/4.0/)
[![Quarto Website](https://img.shields.io/badge/Built_with-Quarto-blue)](https://quarto.org)

Course support website for **MC-451 Research Methods** at Southern Illinois University Edwardsville. The Liaison Program structures a semester-long research project into five progressive phases, guiding students from media consumers to published researchers.

**Live site:** <https://sim-lab-siue.github.io/liaison-program/>

**Instructor:** Dr. Alex P. Leith (aleith@siue.edu)

---

## Site Structure

| Page | Description |
|------|-------------|
| **Home** | Welcome page with course overview and quick-start links |
| **Get Started** | 15-minute workspace setup guide (Obsidian, Git, R, VS Code) |
| **Syllabus** | Full course syllabus with weekly schedule and grading |
| **Phases** | Five progressive modules (see below) |
| **Resources** | Software installation guides, downloadable vault, templates |

### The Five Phases

| Phase | Role | Focus | Weeks |
|-------|------|-------|-------|
| I | The Journalist | Foundations — curiosity, tools, critical reading | 1–4 |
| II | The Architect | Design — theory, prospectus, research questions | 5–7 |
| III | The Builder | Measurement — operationalization, codebook, pilot testing | 8–11 |
| IV | The Analyst | Analysis — data wrangling, visualization, statistical tests | 12–15 |
| V | The Publisher | Communication — portfolio, open science, presentation | 16–17 |

---

## What's Included

- **Installation guides** for R/RStudio, VS Code, Git/GitHub, and Obsidian
- **Liaison Vault** — a downloadable Obsidian workspace with templates for journals, literature notes, codebooks, and project deliverables
- **Weekly assignment templates** aligned with textbook chapters
- **Syllabus contract** and annotated manuscript example (PDFs)
- **Supplemental textbook package** (optional ZIP)

## Building Locally

### Prerequisites

- [Quarto](https://quarto.org/docs/get-started/) (v1.3 or later)

### Render the site

```bash
quarto render
```

Output goes to `docs/`. Open `docs/index.html` in your browser.
